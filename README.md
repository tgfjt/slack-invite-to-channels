# Lacks

## to develop

- Preact: https://preactjs.com/
- CSS: https://bulma.io/
  - Icons: https://material.io/tools/icons/
- Slack API: https://api.slack.com/methods/
- Lorca: https://github.com/zserge/lorca/
- Gin: https://github.com/gin-gonic/gin

To run app:

```console
make dev
```

To build js:

```console
make js
```
